História da web

A história da web começou pela necessidade de desenvolvimento tecnológico por causa da guerra fria.

Havia uma grande necessidade que computadores executassem calculos balísticos cada vez mais rápido,

pois o atraso poderia ser fatal, visto que um humando levaria muito mais tempo para executa-los.

Entendendo Comunicação Cliente x Servidor

A comunicação cliente x servidor, é basicamente uma comunição de chamada, os humanos se comunicam

conversando, e para uma comunição cliente x servidor, não é diferente. Os serviços dos lados do

cliente chamam as requisições dos servidores, e os servidores ao escutarem essas chamadas,

respondem ao clientes entregando uma cópia da requisição.

O que são Clientes?

Os clientes são os usuário que ao ujtilizarem algum serviço de rede, que possa requisitar

algum serviço. Ou seja, os cliente são usuários comuns que usam algum app ou web service

que possa executar chamadas aos servidores.

O que são Servidores?

OS servidores são maquinas que hospedam, normalmente, banco de dados e ou arquivos em uma rede,

os servidores podem variar, sendo locais como as intranets ou como sendo servidores hospedados,

servidores web, por exemplo, os quais possuem toda uma estrutura para entregar os requests dos clientes.
